🔄 Workflow Description

This LangGraph workflow processes financial data through 7 sequential steps:

  • Data Collection: Fetches stock data from Yahoo Finance
  • Technical Analysis: Calculates RSI, moving averages, and trend signals
  • Fundamental Analysis: Analyzes P/E ratios, profitability, and growth metrics
  • Market Sentiment: Uses AI to analyze news and market sentiment
  • Generate Insights: Creates comprehensive analysis using LLM
  • Create Visualizations: Generates price charts with technical indicators
  • Final Recommendations: Provides investment recommendations

Each step builds upon the previous one, creating a comprehensive financial analysis system.
